## âœ¨ Key Feature: Dynamic Personalization

The core feature of this app is its ability to dynamically render a personalized birthday message using the URL path.

**How it works:**
Simply navigate to `/<Name>` to see a custom greeting.

**Examples:**
- `/gokul` -> Displays "HBD GOKUL"
- `/sarah` -> Displays "HBD SARAH"
- `/alex` -> Displays "HBD ALEX"

Depending on the name entered in the URL, the application automatically upcases it and embeds it into the celebratory animation and text.
